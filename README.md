# snake_game

 Flutter as a simple game engine by creating a classic 2D Snake Game.
 Get to know the basics of 2D game graphics and how to control objects.


## What did I learn
* Use Flutter as a game engine
* Move objects
* Control movement
* Build game UI
* Add game elements
